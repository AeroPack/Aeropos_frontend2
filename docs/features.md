# AeroPOS — Feature List

> **For Marketing & Promotions Team**
> This document lists all features available in the AeroPOS app, organized by category. Use this as a reference for designing ads, campaigns, and promotional materials.

---

## 🏪 Point of Sale (POS)

A powerful and flexible checkout system designed for any business type.

| Feature | Description |
|---|---|
| **Multiple POS Layouts** | 5 layouts to match any business: Compact, Retail, Restaurant, Touch, and Dual-Screen |
| **Product Search & Category Filtering** | Instantly search or browse products by category |
| **Cart Management** | Add, remove, and adjust quantities in the cart with real-time totals |
| **Discount & Tax Support** | Apply item-level and order-level discounts and taxes |
| **Multiple Payment Methods** | Accept cash, card, and other payment types |
| **Quick Checkout** | Fast, touch-friendly interface optimized for speed |
| **Hold & Resume Orders** | Put an order on hold and resume it later |
| **Customer Assignment** | Link a sale to a specific customer at checkout |

---

## 📦 Inventory Management

Full control over your products, stock, and catalog.

| Feature | Description |
|---|---|
| **Product Catalog** | Create and manage a full catalog of products with images, SKUs, and pricing |
| **Category Management** | Organize products into categories for easy browsing and filtering |
| **Brand Management** | Track products by brand |
| **Unit Management** | Define units of measure (kg, pcs, liters, etc.) for products |
| **Stock Tracking** | Monitor stock levels in real time |
| **Low Stock Alerts** | Get notified when a product is running low |
| **Inventory Reports** | View detailed reports on stock movement and valuation |
| **Invoice Settings** | Configure invoice numbers, prefixes, and tax settings for the business |

---

## 🧾 Invoicing & Sales History

Professional invoicing with full history and templates.

| Feature | Description |
|---|---|
| **Invoice Generation** | Create professional invoices for every sale |
| **Invoice History** | Browse and search all past invoices |
| **Multiple Invoice Templates** | Choose from 8+ templates: Classic, Modern, Luxury, Dreams, Stylish, Simple, Advanced GST, Thermal |
| **Custom Invoice Template Editor** | Design your own invoice template with a visual editor |
| **Invoice Preview** | Preview invoices before printing or sharing |
| **Print & Share** | Print invoices or share them digitally via PDF |
| **GST / Tax-Ready** | Advanced GST invoice layout with accurate tax breakdowns |

Control your team, roles, and permissions.

## 🏢 Multi-Company Support

Run multiple businesses from a single account.

| Feature | Description |
|---|---|
| **Multiple Companies** | Create and manage multiple business entities under one account |
| **Company Switcher** | Instantly switch between companies without logging out |

## 🔐 Authentication & Security

Enterprise-grade security for your data.

| Feature | Description |
|---|---|
| **Secure Login** | Email & password authentication with JWT tokens |
| **Google Sign-In** | Sign up and log in with a Google account |
| **Multi-Company Login** | Select which company to work in after authenticating |
| **Token-Based Security** | All API calls are secured with authentication tokens |
| **Role-Based Permissions** | Users can only access features their role allows |
